- 每一步先运行新增测试确认其因目标缺陷失败，再实现并重跑同一测试。
- 遵从用户的单提交边界：每个问题只创建一个 GREEN 提交，RED 测试不单独提交。
- 每个代码改动旁保留问题来源和解决办法注释。
- 每个提交只暂存当前问题的测试、实现和跟踪文档。
