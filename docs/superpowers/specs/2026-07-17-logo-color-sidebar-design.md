# Logo 同色系侧栏设计

## 目标

以 SpaceMIT Logo 主绿色 `#B0E237` 建立侧栏色彩关联，同时让侧栏导航在亮色和深色主题下保持可读。

## 方案

- 新增侧栏专用 CSS 变量：背景、边框、常规文字/图标、悬停背景、选中背景和选中文字/图标。
- 亮色主题使用低饱和浅青柠背景，而非直接满铺主绿；选中与悬停状态提高同色系浓度。
- 深色主题使用深橄榄背景，并以浅青柠作为交互强调色。
- `AppLayout` 和 `RouteMenu` 仅消费上述变量，不改变全局主蓝色、内容区或菜单结构。

## 验证

新增应用壳样式合同测试，确认侧栏及菜单使用专用令牌；运行聚焦测试、目标 ESLint 与生产构建。浏览器在真实登录态下的视觉复验作为独立验证边界。
