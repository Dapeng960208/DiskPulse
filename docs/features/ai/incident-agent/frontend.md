# 事件 AI 处置 Agent：前端事实

事件中心页提供“AI 处置设置”入口。设置对话框可启用/关闭 Agent、选择全局已启用模型并用上移/下移确定回退顺序，同时校验 IOPS 绝对下限和基线比例。服务端仍是权限和模型可用性的唯一裁决者，未获超级管理员授权时返回明确错误。

事件列表显示独立的“AI 紧急度”标签；最新 `ai_review.status=running` 时优先显示“AI 审查中”，无研判时显示 `-`。事件详情抽屉展示最近 AI 审查的状态、触发方式和开始/完成时间；有 `ai_assessment` 时再展示分类、紧急度、摘要、研判依据、排查建议、解决方案和最近运行时间。时间线将 `ai_analysis` 与 `ai_status_changed` 显示为 AI 处置操作，和人工评论/状态操作保持区分。
