# 应用壳内置默认头像

## 范围

顶部用户菜单提供四个原创 Q 版动态 GIF，供未配置头像的登录用户使用。该能力仅影响应用壳中的头像展示，不新增用户资料字段、上传入口、接口、权限或数据库变更。

## 展示规则

1. 后端 profile 返回 `avatarUrl` 时始终优先显示该头像。
2. `avatarUrl` 为空时，前端根据用户名或展示名称计算稳定索引，从四个内置 GIF 中选择一个。
3. 同一用户在刷新、重新登录后继续看到同一个默认头像；身份为空时稳定回退到首个头像。

## 资源与验证

资源位于 `frontend/src/assets/images/default-avatars/`，均为 96×96、两帧 GIF，包含眨眼与轻微漂浮动画。单元测试覆盖稳定选择与空身份回退；自动化未使用真实登录态浏览器验证，因此真实 LDAP profile 的头像优先级仍需在部署环境确认。
