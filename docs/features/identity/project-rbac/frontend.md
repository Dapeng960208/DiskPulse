# 项目级 RBAC 与统一审计前端

- 项目详情提供成员和项目审计页签；可见性和操作入口由后端返回的 capabilities 决定。
- 系统管理中的统一审计入口为 `/admin/audit-events` 及详情路由，只对超级管理员展示。
- 项目组和用户目录的配额按钮、容量预测入口及相关页签必须以服务端能力和全局开关为准，前端不推断或提升权限。
- AI 新建会话不提交 `project_id`；页面展示的数据仍由服务端当前项目授权过滤。

前端隐藏、禁用和路由守卫只改善体验，不能替代服务端项目隔离。后端模型和 API 边界见 [RBAC 与审计后端说明](./backend.md)。
