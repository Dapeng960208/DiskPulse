# 统一日期时间范围选择器

`frontend/src/components/form/TimeRangePicker.vue` 是全站范围查询唯一使用的日期时间选择组件。组件内部使用当前用户展示时区的墙上时间，仍以 `YYYY-MM-DD HH:mm:ss` 维护表单值；请求 API 前调用 `frontend/src/utils/datetime.js` 的 `toUtcRange()`，将起止值转换为 RFC 3339 UTC `Z`。后端拒绝没有时区的查询边界。

- 1 天内
- 3 天内
- 1 周内
- 1 个月内
- 3 个月内
- 6 个月内
- 1 年内

实时监控、存储集群健康分析、存储空间监控和统一操作审计均复用该组件。维护窗口等仅填写单个开始或结束时间的表单，不属于范围查询，不使用该组件。用户切换时区后，快捷范围与请求转换必须读取新的当前展示时区。
