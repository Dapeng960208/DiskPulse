# 应用壳布局

## 当前行为

- 右侧工作区由可滚动主内容和普通文档流页脚组成；页脚高度为 `40px`，不覆盖业务内容，也不随内容滚动。
- 应用壳统一提供页面外层间距：桌面为 `16px`，视口不超过 `768px` 时为 `12px`。业务页面不得重复增加等价外层 padding；宽表格只在自身容器内滚动。
- 详情路由在对象加载后补全最后一级面包屑。缺少名称或加载失败时保留稳定的“详情”文本，长名称可截断显示但保留完整 `title`。
- 顶栏优先显示后端提供的 `avatarUrl`；缺失时按用户名或展示名稳定选择内置默认头像。该回退不改变用户资料、权限或接口。

## 边界

应用壳只负责布局、导航层级和头像回退，不改变路由可见性、接口、业务页面权限或数据模型。具体 UI 约束见[前端设计与开发规范](../../../standards/frontend/frontend-design-standard.md)。
