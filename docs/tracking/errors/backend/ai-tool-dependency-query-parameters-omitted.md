# AI 工具遗漏依赖 Query 参数

## 标题

AI 动态工具未注册 FastAPI `Depends` 依赖链中的 Query 参数。

## 错误内容

存储集群健康分析路由把 `start_time`、`end_time` 声明在 `_analytics_time_range` 依赖中，而动态工具生成器只读取路由直接 Query 参数。模型因而看不到时间字段且发送的调用缺少时间范围，内部 ASGI 请求返回 `422`。聊天轨迹此前又把具体失败统一显示为“工具请求失败”，使用户无法判断可修复原因。

## 解决方案

工具生成器递归收集路由及其依赖链的 Path、Query 参数，并保留可选参数的路由默认行为。健康分析缺失任一时间端点时，服务端按当前 UTC 时刻或另一端点补足最近 24 小时范围；工具轨迹只展示白名单中的安全失败原因，其余错误继续使用通用提示。

- 出现次数：14。
