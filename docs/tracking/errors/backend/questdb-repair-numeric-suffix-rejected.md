# QuestDB 修复脚本拒绝纯数字时间后缀

## 错误内容

修复脚本默认生成 `%Y%m%d%H%M%S` 纯数字后缀，却在执行时把后缀本身按完整 SQL 标识符校验，报 `unsafe QuestDB identifier: '20260723141953'`。错误发生在任何 DDL 前。

## 解决方案

不要单独把后缀作为 SQL 标识符校验。先把后缀拼入以字母开头的完整修复表名和备份表名，再对完整名称执行严格标识符校验；空后缀继续拒绝。增加纯数字时间后缀回归测试。
