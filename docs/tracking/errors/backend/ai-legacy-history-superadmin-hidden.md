# AI 历史回复缺少范围标记导致超级管理员被隐藏

## 错误内容

AI 历史助手回复的审计记录缺少 `visibility` 范围标记时，读取逻辑先按未知范围拒绝内容，导致会话所属的超级管理员也只能看到“该历史回复关联的项目权限已失效，内容已隐藏”。

## 解决方案

保持普通用户对未知范围历史的关闭式隐藏；在会话已按 `user_id` 验证归属后，允许配置的超级管理员读取自己会话中的旧未知范围记录。不得以此放宽跨用户会话访问。
