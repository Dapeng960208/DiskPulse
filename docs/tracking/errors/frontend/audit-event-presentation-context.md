# 统一审计时间原样展示且请求/触发方与摘要不明确

## 错误内容

统一操作审计列表和详情直接展示带小数秒的 ISO 时间。存储采集等异步审计没有关联人工用户时，页面仅显示 `-`，无法判断其是系统定时任务；详情还固定显示未采集的 IP、请求地址和资源路径，已存在的执行阶段及采集数量摘要没有被可读地表达。

## 解决方案

统一审计时间格式化为本地 `YYYY-MM-DD HH:mm:ss`。请求/触发方优先显示审计关联的用户；没有用户时按 `actor_type` 表明系统定时任务或系统服务。详情仅显示实际记录的关联字段，并将执行阶段、操作、结果和存储采集摘要转换为中文字段；不从前端推断客户端 IP 或完整请求路径。
