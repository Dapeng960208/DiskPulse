# Mock 未将 `/v1/admin` 资源纳入超级管理员边界

## 错误内容

Mock 网关只拦截 `/admin`，没有拦截版本化的 `/v1/admin` 路径，导致普通演示账户可读取 Incident AI 设置。

## 解决方案

统一将 `/admin` 和 `/v1/admin` 作为系统管理路径，在路由分发前执行超级管理员检查；为版本化 Incident AI 设置补充普通账户拒绝和候选顺序保存测试。
