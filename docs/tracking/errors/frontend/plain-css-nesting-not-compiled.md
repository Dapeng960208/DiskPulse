# 普通 CSS 中的嵌套规则未生效

## 错误内容

存储分布组件在普通 `<style>` 中使用 SCSS 嵌套规则，浏览器不会编译该语法，卡片的 `flex: 1` 未生效。切换到该页签时卡片高度仅 41.6px，ECharts 因容器高度为 0 发出警告。

## 解决方案

将样式块声明为 `lang="scss"`，并通过本地浏览器确认页签与卡片均为 746.4px。
