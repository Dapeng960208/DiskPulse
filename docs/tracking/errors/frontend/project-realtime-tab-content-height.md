# 项目资源实时页签未传递完整内容高度

## 错误内容

项目详情“项目使用实时”或项目组详情“容量趋势”页签没有建立完整的全高 flex 父子链。异步页签、内容区滚动或父级高度重新计算后，趋势图和告警表可能按内容自身高度收缩，导致内容区下方出现大片空白或工作区高度为零。

## 解决方案

从详情页根节点、`ElTabs` 内容、容量页签到 `RealTimePage` 建立全高且可收缩的 flex 链。项目实时页由 `ProjectDiskUsage.vue` 传递高度；项目组详情为容量 `ElTabPane` 设置独立 flex class。通用实时组件的 `fillContent` 模式由此获得稳定的父级高度，工作区可以占满筛选和概要信息后的剩余内容区。
