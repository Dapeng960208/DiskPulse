# 项目详情表格纵向滚动被隐藏导致分页不可达

## 错误内容

项目详情的用户目录、项目组、成员与权限和项目审计数据超过可视高度时，表格容器将纵向溢出隐藏，鼠标滚轮不能继续查看表格内容，底部分页可能被推离可视内容区。成员与权限还没有 20 条默认分页。

## 解决方案

`DataTable.vue` 的表格包装层必须使用受限的 flex 高度、`min-height: 0` 和 `overflow: auto`，使纵向滚动只发生在表格内。项目详情四个数据页签及其内容根节点建立完整的 flex 高度链，筛选区不收缩，数据表占用剩余空间，分页保持在表格卡片底部。成员与权限复用 `DataTable`，按筛选结果以 20 条默认页大小分页。
